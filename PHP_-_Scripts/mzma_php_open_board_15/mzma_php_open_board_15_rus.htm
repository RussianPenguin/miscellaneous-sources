Требования к системе:

 

  • PHP версии 3 или новее.
  • База данных MySQL

 

Кодировка файлов – KOI8-R. Если у вас кодировка Win-1251 и сервер Russian Apache, то не забудьте перекодировать текстовые файлы. Так же если у вас Russian Apache необходимо удалить таг кодировки.

 

Для установки достаточно отредактировать строку 25 файла db_lib.php выставив данные, необходимые для доступа к вашей базе данных (имя хоста базы данных, имя базы данных, имя пользователя и пароль), затем скопировать/загрузить файлы конференции в ваш каталог на сервере и направить ваш броузер на файл create_tables.php, который создаст необходимые таблицы. Направив затем броузер на ваш каталог можете начинать пользоваться.

 

Файл create_tables.php больше не нужен и его можно удалить.

 

Что редактировать в файле db_lib.php:

 

·         $database – имя вашей базы данных.

·         $host – Имя хоста базы данных. Чаще всего база данных бывает установленна на том же компьютере что и сам сервер, по этому по умолчанию стоит значение “localhost”.

·         $user – имя пользователя (логин) доступа к базе данных. Пользователь с этим именем должен иметь разрешение читать, изменять и создавать таблицу для базы данных с именем, указанным в переменной $database.

·         $password – Ваш пароль для доступа к базе данных.

 

Количество тем на одной странице регулируется переменной $per_page в файле index.php3. Переменная $CLUB существует для возможности параллельного существования нескольких конференций.
