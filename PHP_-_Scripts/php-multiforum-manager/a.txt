MultiForum Manager
Форум, не требующий mySQL. При этом, у него нет такого недостатка как например в wwwthreads (ruboard) - т.е. создаются файлы только с данными - без элементов дизайна. Имеется вебинтерфейс для администрирования. Скрипт поддерживает возможность подписки пользователей для получения ими сообщений форума по почте.
