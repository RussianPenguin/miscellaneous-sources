A free Yahoo-like link directory

features:
1) unlimited depth of subcategories
2) unlimited number of categories / links
3) case incensitive search of the links database
4) multilanguage support (English, French, Russian, Spanish, Catalan,Turkish in v1.1) for the end-user interface
5) visitors can add links & categories
6) customisable header & footer
7) easy admin interface

changes:
v1.2 - changes in the admin section, made an install script
v1.1 - changes in the admin section, eliminated some MySQL-related bugs
v1.0 - first release

requirements: PHP3 / MySQL / web server

was tested on:
Win98/PHP 3.0.6/MySQL 3.19/Apache 1.3.6
FreeBSD/PHP 3.0.13/MySQL 3.22/Apache 1.3.9

Каталог ссылок

возможности:
1) неограниченная глубина подкатегорий
2) неограниченное количество ссылок/категорий
3) поиск по базе ссылок без учета регистра
4) многоязычный интерфейс для конечного пользователя. (Русский, английский, французский, испанский, каталанский, турецкий)
5) посетители могут добавлять ссылки и категории
6) изменяемый заголовок и конец файла
7) легкий интерфейс одобрения новых ссылок

Для работы программы необходим PHP3 и СУБД MySQL.

программа была испытана на:
Win98/PHP 3.0.6/MySQL 3.19/Apache 1.3.6
FreeBSD/PHP 3.0.13/MySQL 3.22/Apache 1.3.9
