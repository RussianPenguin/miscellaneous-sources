GraphFX - Оболочка для рисования гистограмм

Автор: Norm Almond

Скачать демонстрашку - 192 Kb

Описание

Пример показывает рисование давольно красивых гистограмм, очень похожих на те, которые рисуют в журналах при сравнении производительности процессоров. На вид - много всяких возможностей.

Компилятор: Visual C++.

Использование

Для использования оболочки CGraphFX , включите в проект файлы GraphFX.h, GraphFX.cpp, Screen.h, Screen.cpp, ItemArray.h, ItemArray.cpp, Item.h и Item.cpp.

Далее включите в заголовочный файл отображения "GraphViewFX.h"

#include "GraphViewFX.h"

Потом наследуйте класс отображения от CGraphViewFX.

class CGraphView : public CGraphViewFX

Этого достаточно.

Для примера был использован произвольный массив значений.

Функция OnDraw в Вашем классе отображения должна сперва вызывать базовую CGraphViewFX::OnDraw .
