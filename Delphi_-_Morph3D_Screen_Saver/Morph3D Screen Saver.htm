Morph3D Screen Saver.

Прислал: Чернов Сергей  ( morph3d.nm.ru )

Компилятор: Delphi 4,5,6

Morph3D Screen Saver - это скринсейвер для Windows. При запуске заставки на экране изображаются различные трехмерные вращающиеся объекты. Всего их семнадцать, в их список входят сферы, додекаэдр, икосаэдр, тор, спираль, куб, тетраэдр и другие. Все объекты состоят из 200 светящихся точек. Один объект периодически превращается в другой (эффект "morphing"). Программа написана с использованием функций WinApi, поэтому компилится под любой 32-битный Delphi.
