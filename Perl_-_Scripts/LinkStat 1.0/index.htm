Скрипт статистики обращения по ссылкам

Для сохранения статистики замените ссылки на следующий вид: "/cgi-bin/links.cgi?ВАША_ССЫЛКА"

Пример ссылки

Для получения статистики введите: "/cgi-bin/links.cgi"

Для очистки статистики удалите файл "links.txt" в каталоге /cgi-bin/

Новые версии и другие скрипты доступны по адресу: "http://allbbs.virtualave.net/scripts.htm"
