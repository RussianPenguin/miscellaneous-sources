LinkStat 1.0


Написано на Perl 5, CSS. Статистика сортируется по убыванию количества переходов по ссылкам. Вам не надо что-либо добавлять в файлы базы, добавление ссылки происходит автоматически. В статистике содержится дата и время первого и последнего обращения по ссылке. Вызов ссылки с сохранением статистики выглядит так: '/cgi-bin/links.cgi?ВАША_ССЫЛКА'. Все скрипты содержат комментарии на русском языке.
