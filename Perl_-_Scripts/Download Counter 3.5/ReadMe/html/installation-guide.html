How to use it?

The script is called by creating a link like this
<a href="http://www.yourdomain.com/cgi-bin/clickscounter.pl?url=www.thatdomain.com">Your Link </a>

To access the log file on online you go to the url <a href="http://www.yourdomain.com/cgi-bin/clickcounter.pl?view=password">Your Link</a>

  • You need to change the url of the script clickscounter.pl to where it is located on your server.
  • The url can contain the protocol name (i.e. http://, ftp://, etc). If it does not, the HTTP protocol will be assumed.
  • Change "password" to the password you chose for $view_log

When visitors download files or click on links, script will increase statistic in simple textual-database file

 
How to install it on my server?
 
  1. Copy clickscounter.pl and clicks-counter.log to your cgi-bin directory.
  2. Change attributes of .pl files to executable (chmod 755).
  3. Change attributes of database file (clicks-counter.log) to 'read/write' (chmod 666).
  4. Change your download links as described in 'How to use it?' instruction.
  5. Enjoy! :-) or... if you have any problems contact us!
