General Information

Clicks Counter: Download Counter is a program that will keep track of all hits and clicks on any links of your Web pages. It includes an online viewer, with color tables, to view the log online. Password changeable for viewer. It will run on NT or Unix and requires minimal setup. Also has a lock routine to make sure file does not get erased by heavily traveled sites. It now supports Internet Information Server and Personal Web Server by Microsoft.
