Описание автора:
Collie Perl Shell - интегрированная среда разработки(IDE) для языка Perl. Программа поддерживает работу одновременно с несколькими документами, проектами, имеется подсветка синтаксиса. В программе предусмотрена возможность сохранения файлов с различными форматами конца строки (UNIX,Mac,Win). Имеется встроенный редактор Regular Expressions, утилита CodeLibrary позволяет хранить часто используемые фрагменты кода, CGI wizard автоматически создает заготовку для CGI script'а.
