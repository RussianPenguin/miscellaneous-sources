Расширение к Task Manager (NT)

Автор: Zoltan Csizmadia.

Нажмите на картинку для увеличения

Описание

Данное расширение для Windows Task Manager (NT/2000) обладает следующими возможностями:

  • Показывает иконки процессов (если доступно)
  • Использует различные цвета для сервисов
  • Ищет файлы используемые процессами. (Используйте * для поиска всех используемых файлов)
  • Ищет используемые модули процессом
    (например "kernel") 
  • Показывает ID процесса в Applications tab
  • Использует различные цвета для процессов, если CPU используется на 25%, 50%, 75%
  • Закрывает используемые файлы (Вы можете его удалить, если он был открыт эксклюзивно)
  • Выгружает используемые модули (опять же, чтобы его можно было удалить)
  • Так же можно убивать сервисы !!!!
  • Убивать процессы одним нажатием кнопки DEL !

Для большей производительности Вам лучше заходить в систему как administrator!

Использование

Скопируйте TaskManagerEx.exe и TaskManagerEx.Dll в любую директорию, а потом запустите TaskManagerEx.exe. Он будет перехватывать Task Manager.

Внимание

Если Вы выгрузите или закроете файл/модуль у процесса, то он может разрушиться.
