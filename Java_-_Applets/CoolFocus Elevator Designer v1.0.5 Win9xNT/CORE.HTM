CoolFocus Elevator Designer v1.0.5

Menu applets, by definition, have a lot of tags to be
entered, which takes a lot of precise typing and patience. It's a
laborious job, particularly with some of our heavyweight systems..
like Elevator. Elevator Designer aims to take the pain out of the
job by providing a simple step-by-step wizard-style creation
system for placing Elevator on a web page. Just work through the
7 pages selecting the options you want, adding and arranging items
graphically on the tree, and selecting documents to link to by
choosing them from a standard file dialog. A quick keystroke or
button click previews the result in the built-in browser (or any
external browser) and you can continue to click that button after
any change to update the browser. When you have everything set up
as you'd like it, Elevator Designer will create the entire HTML
code for you automatically and either copy it to the clipboard (so
that you can paste it into an existing web page) or create a web
page for you. You can even open a web page and have the code
inserted at the point you choose. Best of all, Elevator Designer
can read your web page at any time and reload all your settings
from it, making it simple to change colors and add, edit or
remove menu entries in seconds.
