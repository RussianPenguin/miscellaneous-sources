Описание

 Используя этот апплет можно создавать разнообразные горизонтальные линии на сайте. Меняя параметры апплета, можно создать линии соответствующую дизайну сайта. Для применения этого апплета на своём сайте скопируйте rainbow.class в свою основную директорию и вставьте в HTML следующий код.

 
 Код


 
 Параметры апплета
  1. <APPLET CODE="rainbow.class" Запуск апплета.
  2. WIDTH=650 HEIGHT=5> Высота и ширина линии.
  3. <PARAM NAME="squares" VALUE="50"> Количество квадратов в линии.
  4. <PARAM NAME="step" VALUE="10"> Шаги смены цвета между квадратами.
  5. <PARAM NAME="speed" VALUE="20"> Скорость в милисекундах.
  6. <PARAM NAME="color" VALUE="green"> Выбор цвета green , red или blue .
  7. <PARAM NAME="horizontal" VALUE="-1"> VALUE может быть 1 или -1 .
    1 для движения цвета вправо
    -1 для движения цвета влево.
  8. </APPLET> Тег завершения кода апплета
