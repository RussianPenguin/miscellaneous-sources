MenuScroll is designed to simulate a windows pop-up menu (the kind of menu that appears when you right click on your desktop), and is
used for site navigation (but with a little imagination it could also be used for presentation). The applet will appear as two menus, the left menu (main) will always be visible and drive all the other (sub) menus.

Install Notes
Unzip, unrar and you are ready to use this fine release. By the way, with this release, you don't need to include the following parameter
line in your .html files: <param name=copyright value="Menu Scroll v1.0 Copyright (c) 1999, consulting.com Inc.">
