Описание

Этот апплет создает 3D часы вращающиеся в пространстве. С помощью специальных параметров можно изменять внешний вид апплета. Для использования возьмите файл Clock3D.class .

Параметры:
FPS
Количество фреймов в секунду (по умолчанию 13).
STYLE
Стиль представления часов: 1 - вращающиеся; 2 - скручивающиеся (по умолчанию 1).
12HOUR
12-и часовые часы - 1, 24-х часовые - 0 (по умолчанию 0).
PIXD
Расстояние между двумя точками. При STYLE=1 - по вертикали, при STYLE=2 - по вертикали и горизонтали (по умолчанию 16).
PIXANGLE
Для STYLE=1 угол между думяа горизонтально-соседними точками. Для STYLE=2 степень "скручивания" между двумя точками (по умолчанию 8).
RADIUS
Для STYLE=1 радиус окружности вращения (по умолчанию 15).
A1
Дистанция между часами и экраном монитора, чем она меньше, тем изобоажение больше (по умолчанию 5000).
ROTX
Скорость вращения вокруг оси X (по умолчанию 0).
ROTY
Скорость вращения вокруг оси Y (по умолчанию 4).
ROTZ
Скорость вращения вокруг оси Z (по умолчанию 0).
IROTX
Угол вращения (циклического сдвига) вокруг оси X (по умолчанию 0).
IROTY
Угол вращения (циклического сдвига) вокруг оси Y (по умолчанию 0).
IROTZ
Угол вращения (циклического сдвига) вокруг оси Z (по умолчанию 0).
COLOR
Цвет точек (по умолчанию #FF0000 - красный).
BGCOLOR
Цвет фона (по умолчанию #000000 - черный).
